# Project Detail

**Parent:** [web-app](../README.md)

Tabbed hub for managing a single project.

- **Type**: composed
- **Route**: `/project-detail/index.html`
- **Params**: `?projectId=`

## Features

- Tabbed interface: Tasks, Discussion, History, Data
- Task list with status tracking
- Team discussion thread
- Version history timeline
- Project metrics and milestone progress
