# Approval Detail

**Parent:** [web-app](../README.md)

Deep review page for approving or rejecting an idea.

- **Type**: standalone
- **Route**: `/approval-detail/index.html`
- **Params**: `?ideaId=`

## Features

- Full idea details with problem/solution/outcome
- Edge definition data and metrics
- Scoring breakdown and risk assessment
- Approve and reject action buttons
- Clarification Q&A thread
