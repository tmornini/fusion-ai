# Fonts

Self-hosted woff2 font files loaded via `@font-face` declarations in `../styles/fonts.css`.

## Font Files

| File | Family | Weight |
|------|--------|--------|
| `ibm-plex-sans-400.woff2` | IBM Plex Sans | 400 (Regular) |
| `ibm-plex-sans-500.woff2` | IBM Plex Sans | 500 (Medium) |
| `ibm-plex-sans-600.woff2` | IBM Plex Sans | 600 (Semibold) |
| `ibm-plex-sans-700.woff2` | IBM Plex Sans | 700 (Bold) |
| `inter-400.woff2` | Inter | 400 (Regular) |
| `inter-500.woff2` | Inter | 500 (Medium) |
| `inter-600.woff2` | Inter | 600 (Semibold) |
| `inter-700.woff2` | Inter | 700 (Bold) |
| `ibm-plex-mono-400.woff2` | IBM Plex Mono | 400 (Regular) |

## Usage

- **IBM Plex Sans** — Display headings and titles
- **Inter** — Body text and UI elements
- **IBM Plex Mono** — Code and data values
